# MinecraftBut
A Minecraft Plugin for youtubers such as BajanCandian, Skeppy, BadBoyHalo, A6D
The youtubers will spawn in a small world with the option to apply multiple different senerarios

## Senerario
* Lava is Rising
* Water is Rising
* Raining TNT
* Raining Blocks
* No Jumping
* Dragon spawns
* One heart
